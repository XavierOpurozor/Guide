Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of a webpage, while CSS handles the styling and JavaScript adds interactivity.

This tutorial covers HTML5, the latest version of the HTML standard. We'll explore the core concepts and dive into more advanced topics.

Basic Structure of an HTML Document

Every HTML document follows a standard structure: 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <!-- Content of the webpage goes here -->
    <h1>Hello, World!</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

Let's break down each part:

  • <!DOCTYPE html>: The document type declaration. It tells the browser which version of HTML the page is written in. For HTML5, it's simply <!DOCTYPE html>.
  • <html>: The root element of the HTML page. The lang attribute specifies the language of the document (e.g., "en" for English). This is important for accessibility and search engines.
  • <head>: Contains meta-information about the document, such as the title, character set, and viewport settings. This information is not displayed on the page itself.
  • <body>: Contains all the visible content of the webpage, including text, images, links, and other elements.

Headings

HTML headings are used to structure content. There are six levels of headings, from <h1> to <h6>. <h1> is the most important heading, while <h6> is the least. 


<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>
<h4>Subsection Heading</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

Proper use of headings is crucial for SEO and accessibility. Use headings to create a clear document outline.

Paragraphs

The <p> tag is used for paragraphs of text. Browsers automatically add some space before and after each paragraph. 


<p>This is the first paragraph.  It can contain multiple sentences.  The text will wrap automatically to fit the width of the browser window.</p>
<p>This is another paragraph.  Each paragraph should represent a distinct block of text.</p>

Text Formatting

HTML provides several tags for formatting text:

  • <strong>: Important text, usually displayed in bold.
  • <em>: Emphasized text, usually displayed in italic.
  • <br>: Inserts a single line break. Use this sparingly; use paragraphs for larger blocks of text.
  • <hr>: Creates a horizontal rule (line) across the page.
  • <mark>: Marks highlighted text.
  • <small>: Defines smaller text.
  • <del>: Represents deleted text.
  • <ins>: Represents inserted text.
  • <sub>: Subscript text.
  • <sup>: Superscript text.

<p><strong>This is important text</strong>.</p>
<p><em>This text is emphasized</em>.</p>
<p>This is a line<br>with a line break.</p>
<hr>
<p>This is <mark>highlighted</mark> text.</p>
<p><small>This is smaller text.</small></p>
<p><del>Deleted text</del> <ins>Inserted text</ins></p>
<p>This is <sub>subscript</sub> and <sup>superscript</sup> text.</p>

Links

The <a> tag is used to create hyperlinks. Links can point to other web pages, files, or locations within the same page. 


<a href="https://www.example.com">Visit Example.com</a>
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Visit Example (new tab)</a>
<a href="/about.html">About Us</a> <!-- Relative link -->
<a href="#section3">Jump to Section 3</a> <!-- Link to an ID on the same page -->
<a href="mailto:info@example.com">Email Us</a>

Attributes:

  • href: Specifies the URL of the link. It can be an absolute URL (like "https://www.example.com"), a relative URL (like "/about.html"), a link to an element on the same page (like "#section3"), or a mailto link ("mailto:info@example.com").
  • target: Specifies where to open the linked document. Common values include _blank (new tab or window), _self (default, opens in the same tab), _parent, and _top (used with frames).
  • rel: Specifies the relationship between the current document and the linked document. noopener and noreferrer are often used with target="_blank" for security reasons.

Images

The <img> tag is used to embed images in a web page. Images can improve the visual appeal of a page and convey information. 


<img src="images/myimage.jpg" alt="Description of the image" width="300" height="200">

Attributes:

  • src: Specifies the path to the image file. The path can be relative or absolute.
  • alt: Provides alternative text for the image. This is essential for accessibility (screen readers) and if the image fails to load. It's also good for SEO.
  • width and height: Specify the dimensions of the image in pixels. It's a good practice to include these to prevent layout shifts while the image is loading. You can also use CSS to style the image dimensions.

Lists

HTML supports ordered lists, unordered lists, and definition lists. Lists are used to structure content in a clear and organized way.

Unordered Lists (<ul>)

Unordered lists are used for items that do not have a specific order. They are typically displayed with bullet points. 


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered Lists (<ol>)

Ordered lists are used for items that have a specific order. They are typically displayed with numbers. 


<ol>
    <li>First step</li>
    <li>Second step</li>
    <li>Third step</li>
</ol>

Definition Lists (<dl>)

Definition lists are used to define terms and their descriptions. 


<dl>
  <dt>HTML</dt>
  <dd>Hypertext Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Lists can be nested within each other to create hierarchical structures. 


<ul>
  <li>Item 1
    <ul>
      <li>Sub-item 1.1</li>
      <li>Sub-item 1.2</li>
    </ul>
  </li>
  <li>Item 2</li>
</ul>

Tables

The <table> tag is used to create tables. Tables are used to display data in a structured format. 


<table>
  <caption>Sample Data Table</caption>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John Doe</td>
      <td>30</td>
      <td>New York</td>
    </tr>
    <tr>
      <td>Jane Smith</td>
      <td>25</td>
      <td>Los Angeles</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
        Source: Example Data
    </tr>
  </tfoot>
</table>

Key tags:

  • <table>: The table element.
  • <caption>: Adds a caption to the table.
  • <thead>: The table header.
  • <tbody>: The table body.
  • <tfoot>: The table footer
  • <tr>: Table row.
  • <th>: Table header cell.
  • <td>: Table data cell.
  • colspan: Attribute to make a cell span across multiple columns.
  • rowspan: Attribute to make a cell span across multiple rows.

Forms

The <form> tag is used to create HTML forms for user input. Forms are used to collect data from users, such as names, email addresses, and passwords. 


<form action="/submit-form" method="post">
  <fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" placeholder="Enter your name" required><br>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email" placeholder="Enter your email" required><br>

    <label for="password">Password:</label>
    <input type="password" id="password" name="password" placeholder="Enter your password" required minlength="8"><br>
  </fieldset>

  <fieldset>
    <legend>Preferences</legend>
    <label>Gender:</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br>

    <label for="subscribe">Subscribe to newsletter:</label>
    <input type="checkbox" id="subscribe" name="subscribe" checked><br>

    <label for="cars">Choose a car:</label>
    <select id="cars" name="cars">
      <option value="volvo">Volvo</option>
      <option value="saab">Saab</option>
      <option value="mercedes">Mercedes</option>
      <option value="audi">Audi</option>
    </select><br>
  </fieldset>

  <label for="message">Message:</label><br>
  <textarea id="message" name="message" rows="4" cols="50" placeholder="Enter your message"></textarea><br>

  <button type="submit">Submit</button>
  <button type="reset">Reset</button>
</form>

Common input types and attributes:

  • text: Single-line text input. The placeholder attribute provides a hint to the user. The required attribute makes the field mandatory.
  • email: Input field for email addresses. The browser will validate the input.
  • password: Password input. The input is masked. The minlength attribute specifies the minimum number of characters.
  • radio: Radio buttons. Used to select one option from a group. The name attribute must be the same for all radio buttons in the group.
  • checkbox: Checkboxes. Used to select one or more options. The checked attribute makes the checkbox selected by default.
  • submit: Submit button. Sends the form data to the server.
  • reset: Reset button. Resets the form fields to their initial values.
  • textarea: Multi-line text input. The rows and cols attributes specify the size of the textarea.
  • select: Dropdown list. The <option> tags define the options in the list.

Other form related elements:

  • <fieldset>: Groups related elements in a form.
  • <legend>: Defines a caption for the <fieldset> element.

Div and Span

<div> and <span> are important container elements used for grouping and styling other HTML elements.

  • <div> (division): A block-level element used to group other HTML elements. It creates a line break before and after the div. It's commonly used for layout purposes.
  • <span>: An inline element used to group inline elements. It does not create line breaks.It's often used to style specific parts of text.

<div style="background-color:lightblue; padding:20px; border-radius: 8px;">
  <p>This is a div element.  It acts as a container for other elements.  You can apply styles to the div using CSS.</p>
  <span style="color:red; font-weight:bold;">This is a span element</span> inside the div.  It allows you to style specific parts of the text without creating a new block.<br>
  <span style="font-size: 1.2em;">Another span element</span> with different styling.
</div>

Semantic HTML

Semantic HTML uses tags to convey the meaning of the content, not just its appearance. This improves accessibility, SEO, and makes the code more readable and maintainable.

Examples of semantic elements:

  • <header>: A header for a document or section. It typically contains the main heading, logo, and navigation.
  • <nav>: Navigation links.
  • <main>: The main content of a document.
  • <article>: An independent, self-contained piece of content, such as a blog post or news article.
  • <section>: A thematic grouping of content within a document.
  • <aside>: Content related to the main content, but not essential to it, such as a sidebar.
  • <footer>: A footer for a document or section. It typically contains copyright information, contact information, and links to related documents.

<header>
  <h1>My Website</h1>
  <nav>
    <ul>
      <li><a href="/">Home</a></li>
      <li><a href="/about">About</a></li>
      <li><a href="/contact">Contact</a></li>
    </ul>
  </nav>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <p>This is the content of the article.  It's a self-contained piece of content.</p>
  </article>
  <section>
    <h2>Section Heading</h2>
    <p>Content of the section.  This section groups related content.</p>
  </section>
  <aside>
    <p>This is a sidebar with related information.</p>
  </aside>
</main>

<footer>
  <p>© 2025 My Website | <a href="/privacy">Privacy Policy</a> | <a href="/terms">Terms of Service</a></p>
</footer>

Embedding Media

You can embed various media in your HTML pages.

Images (covered before)

The <img> tag is used to embed images.

Audio

The <audio> tag is used to embed audio content. 


<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Video

The <video> tag is used to embed video content. 


<video controls width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>

Embedding YouTube Videos

You can also embed videos from platforms like YouTube using an <iframe>.
