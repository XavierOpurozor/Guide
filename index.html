1. What is a <header>?

The <header> tag is a semantic HTML element that typically contains introductory content like the website logo, navigation, and welcome messages.

2. How to Use <header>

It should be placed at the top of the page or inside sections to describe their heading area.

<header>
  <h1>Welcome to NEXH</h1>
</header>

3. Styling Headers

Headers can be styled with background colors, shadows, and padding to stand out.

4. What is <nav>?

<nav> stands for Navigation. It wraps around main navigation links in a webpage, improving SEO and accessibility.

5. Using <nav> for Menus

<nav>
  <a href="#home">Home</a>
  <a href="#contact">Contact</a>
</nav>

6. Styling <nav>

Flexbox is ideal for aligning nav links horizontally. Add spacing, hover effects, and responsiveness.

7. What is Padding?

Padding is the space between an elementâ€™s content and its border.

It helps in making content breathable inside boxes.

8. Example: Padding

div {
  padding: 20px;
}

9. Padding Shorthand

Use 1 to 4 values:

padding: 10px 20px 15px 5px; /* top, right, bottom, left */

10. What is Margin?

Margin is the space around the outside of elements. It separates components visually.

11. Using Margin

div {
  margin: 20px;
}

12. Collapsing Margins

Vertical margins between two block elements may collapse into one. Be aware!

13. What is <footer>?

The <footer> tag is used to define the bottom section of a page, including credits, links, or contact info.

14. Footer Example

<footer>
  © 2025 NEXH. All rights reserved.
</footer>

15. Styling Footers

Use subtle colors and small fonts. It should feel like the end of the document, not a wall.
